## D1 Mini Air Quality Monitor
This is a DIY ESP8266 based air quality monitor with a CO2 and PM2.5 sensor, which displays the information over WiFi on a local web interface.

### Features:
* CO2 Monitoring (ppm)
* PM2.5 monitoring (ug/m3)
* Web interface accessible on local devices
* Real-time information updates
* Measurements are shown beside recommended Air Quality Index levels


### Hardware required:
 * LOLIN Wemos D1 Mini V4.0.0
 * SenseAir S8 CO2 Sensor
 * Plantower PMS5003 Particulate Sensor
